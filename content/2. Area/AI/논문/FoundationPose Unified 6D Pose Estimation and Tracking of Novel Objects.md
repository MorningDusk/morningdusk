# 1. Introduction
## 1.1. 연구 배경과 문제 정의
**6D 자세 추정**<sub>6D pose estimation</sub>은 물체의 3차원 위치 $(x,y,z)$와 3차원 회전 $(roll, pitch, yaw)$을 모두 추정하는 핵심적인 컴퓨터 비전 기술이다. 이는 로봇 조작, 가상 현실, 자율 주행 등 다양한 실제 응용에서 필수적인 기능이다. 기존 연구들은 다음과 같은 근본적인 한계점을 가지고 있다.
- **인스턴스 수준 방법**<sub>instance-level method</sub>: 학습 시점에 결정된 특정 물체 인스턴스에만 작동한다. 새로운 물체에는 적용할 수 없고, 일반적으로 질감이 있는 CAD 모델이 필요하다.
- **카테고리 수준 방법**<sub>category-level method</sub>: 미리 정의된 카테고리 내의 물체들만 처리 가능하다. 카테고리 수준의 학습 데이터 확보가 매우 어렵고, 자세 정규화<sub>pose canonicalization</sub> 과정이 복잡하다.
- **설정 분리 문제**: 모델 기반<sub>model-based</sub>(CAD 모델 제공)과 모델 프리<sub>model-free</sub>(참조 이미지 제공) 설정이 완전히 분리되어 있어 실제 응용에서 유연성이 부족하다.

## 1.2. 연구의 기여점
이 논문은 **FoundationPose**라는 통합 foundation model을 제안하여 위의 문제들을 해결한다.
1. **통합 프레임워크**: 처음 보는 새로운 물체에 대해 모델 기반과 모델 프리 설정을 모두 지원하는 단일 시스템을 구축했다.
2. **다중 작업 지원**: 단일 프레임에서 자세 추정과 비디오 시퀀스에서 추적을 동시에 수행할 수 있다.
3. **강력한 일반화**: 대규모 학습 데이터 학습과 LLM을 활용하여 강력한 일반화 성능을 달성했다.
4. **최고 수준 성능**: 각 작업별로 특화된 기존 방법들을 크게 앞서는 state-of-the-art 성능을 달성했다.
# 2. Related Work
## 2.1. CAD 모델 기반 물체 자세 추정
인스턴스 수준 방법들은 6D 자세 추정의 전통적인 접근법으로, 특정 물체의 질감 있는 CAD 모델을 필요로 한다. 이들의 주요 접근법은 다음 세 가지로 나뉜다:
1. **직접 회귀**<sub>direct regression</sub>: 신경망이 직접 6D 자세를 예측하는 방식이다.
2. **2D-3D 대응**<sub>2D-3D correspondance</sub>: 2D 키포인트와 3D 모델 점을 매칭한 후 PnP 해법을 이용한다.
3. **3D-3D 대응**<sub>3D-3D correspondance</sub>: 점군 간 매칭 후 최소제곱 맞춤<sub>least squares fitting</sub>을 수행한다.
카테고리 수준 방법들은 같은 카테고리 내의 새로운 인스턴스에 적용할 수 있지만, 여전히 미리 정의된 카테고리를 벗어날 수 없다는 제약이 있다. 최근 연구들은 임의의 새로운 물체에 대해 CAD 모델만 주어지면 즉시 적용 가능한 방법들을 개발하고 있다.
## 2.2. Few-shot 모델 프리 물체 자세 추정
모델 프리 방법들은 CAD 모델 대신 참조 이미지 몇 장만으로 물체를 인식하려는 시도들이다:
- **RLLG와 NeRF-Pose**: 인스턴스별 학습을 수행하지만 CAD 모델 없이 학습하는 방법을 제안했다. 
- **Gen6D**: 검출, 검색, 정제 파이프라인을 사용하지만, 분포 밖 문제로 인해 미세 조정이 필요하다.
- **OnePose 시리즈**: 움직임으로부터 구조<sub>Structure-from-Motion</sub>를 활용한 물체 모델링과 2D-3D 매칭 네트워크를 사용한다.
- **FS6D**: RGBD 정보를 활용하지만 여전히 대상 데이터셋에서 미세 조정이 필요하다.
이들의 공통적인 한계점은 대부분 대응<sub>correspondance</sub>에 의존한다는 것이다. 이로 인해 질감이 없는 물체나 심한 가림 상황에서 매우 취약하다.
## 2.3. 물체 자세 추적
**물체 자세 추적**<sub>object pose tracking</sub>은 시간적 정보를 활용해 더 효율적이고 부드러운 자세 예측을 수행하는 분야이다. 기존 추적 방법들의 문제점은 대부분 자세 추정과 분리되어 있어서 초기 자세를 외부에서 제공받아야 한다는 것이다.
# 3. Approach
## 3.1. 언어 지원 대규모 데이터 생성

- **Objaverse**: 1156개 LVIS 카테고리에 속하는 4만개 이상의 일상 물체들
- **GSO**: 고품질로 스캔된 일상 물체들

### LLM 지원 질감 증강

## 3.2. 신경 물체 모델링
### 필드 표현의 핵심 설계
모델 프리 설정에서 3D CAD 모델이 없을 때 핵심적인 도전은 물체를 효과적으로 표현하여 하위 모듈에 충분한 품질의 이미지를 렌더링하는 것이다. **신경 암시적 표현**<sub>neural implicit representation</sub>은 새로운 시점 합성에 효과적이면서 GPU에서 병렬 처리가 가능하다. 이 연구에서는 물체 중심 신경 필드 표현을 도입했다:
- **기하 함수** $\Omega: x \to s$는 3D점 $x \in \mathbb{R}^3$을 입력으로 받아 부호 거리 값 $s\in\mathbb{R}$을 출력한다.
- **외관 함수**: $\Phi: (f_\Omega (x), n, d) \to c$는 기하 네트워크의 중간 특징 벡터, 점 법선, 시점 방향을 입력으로 받아 색상을 출력한다.
암시적 물체 표면은 부호 거리 필드의 영 수준 집합으로 정의된다:
$$
S = \{x\in \mathbb{R}^3|\Omega (x)=0\}
$$
![[3. Resource/Image/Pasted image 20250728123602.png]]
### 필드 학습의 수학적 기초
질감 학습을 위해 절단된 표면 근처에서 체적 렌더링을 따른다. 색상 렌더링은 다음과 같이 정의된다:
$$
c(r)=\int^{z(r)+0.5\lambda}_{z(r)-\lambda}w(x_i)\Phi (f_{\Omega(x_i)}, n(x_i),d(x_i))dt
$$
여기서 $w(x_i)$는 종 모양 확률 밀도 합수로 다음과 같다:
$$
w(x_i)=\frac{1}{1+e^{\alpha\Omega (x_i)}} \frac{1}{1+e^{\alpha\Omega (x_i)}}
$$
학습 과정에서는 여러 손실 함수를 사용한다:
- **색상 감독 손실**: $\mathcal{L}_c=\frac{1}{|R|}\sum_{r\in R} \| c(r)-\bar{c}(r)\|_2$
- **빈 공간 손실**: $\mathcal{L}_e=\frac{1}{|X_e|} \sum_{x \in X_e} |\Omega(x)-\lambda|$
- **표면 근처 손실**: $\mathcal{L}_s=\frac{1}{X_s} \sum_{x\in X_s}(\Omega(x)+d_x-d_D)^2$
## 3.3. 자세 가설 생성
### 자세 초기화 전략
RGBD 이미지가 주어지면 Mask R-CNN이나 CNOS 같은 기성 방법을 사용하여 물체를 검출한다.
- **평행이동 초기화**: 검출된 2D 경계 상자 내의 중간 깊이에 위치한 3D 점을 사용한다.
- **회전 초기화**: 물체를 중심으로 하는 이십면체에서 $N_s=42$개의 시점을 균등하게 샘플링한다.
이런 카메라 자세들은 $N_i=12$개의 이산화된 평면 내 회전으로 추가 증강되어, 총 $N_s \times N_i = 504$개의 전역 자세 초기화가 자세 정제기의 입력으로 전달된다.
![[3. Resource/Image/Pasted image 20250728123640.png]]
### 자세 정제 네트워크 구조
자세 정제 네트워크는 거친 자세에 조건화된 물체의 렌더링과 카메라에서 관찰된 입력의 자르기를 입력으로 받아 자세 품질을 개선하는 자세 업데이트를 출력한다. 네트워크는 평행 이동 업데이트 $\Delta t \in \mathbb{R}^3$과 회전 업데이트 $\Delta R \in SO(3)$을 예측한다. 입력 거친 자세 $[R|t]\in SE(3)$는 다음과 같이 업데이트된다:
$$
\begin{align}
t^+=t+\Delta t \\ R^+=\Delta R \otimes R
\end{align}
$$
네트워크 학습은 L2 손실로 감독된다:
$$
\mathcal{L}_{refine}=w_1 \|\Delta t-\Delta \bar{t}\|_2 + w_2\|\Delta R-\Delta\bar{R}\|_2
$$
## 3.4. 자세 선택
### 계층적 비교의 2단계 전략
정제된 자세 가설들의 리스트가 주어지면 계층적 자세 순위 네트워크를 사용하여 이들의 점수를 계산한다. 네트워크는 2단계 비교 전략을 사용한다.
1. 각 자세 가설에 대해 렌더링된 이미지가 자른 입력 관찰과 비교된다.
2. 모든 $K$개 자세 가설 간의 두 번째 수준 비교를 도입한다.
![[3. Resource/Image/Pasted image 20250728123650.png]]
### 대비 검증 학습
자세 순위 네트워크를 학습하기 위해 자세 조건화된 삼중항 손실을 제안한다:
$$
\mathcal{L}(i^+,i^-)=\max(S(i^-)-S(i^+)+\alpha,0)
$$
여기서 $\alpha$는 대조 여백이고, $i^-$와 $i^+$는 각각 음성과 양성 자세 샘플을 나타낸다.
# 4. Experiment
## 4.1. 데이터셋 및 실험 설정
### 평가 데이터셋들
5개의 도전적인 데이터셋에서 종합적인 평가를 수행했다:

- **T-LESS**: 30개의 질감 없는 산업용 물체들
- **YCB-InEOAT**: 로봇 조작 중의 동적 외부 가림

### 작업 조합
프레임워크가 통합되어 있기 때문에 두 설정(모델 프리와 모델 기반)과 두 자세 예측 작업(6D 자세 추정과 추적)의 조합을 고려하여 총 4가지 작업을 모두 평가했다.
## 4.2. 자세 추정 결과
### 모델 프리 설정 성능
**YCB-Video 데이터셋**에서 FoundationPose는 97.4% ADD-S와 91.5% ADD를 달성하여 기존 최고 성능인 FS6D의 88.4% ADD-S, 42.1% ADD를 크게 앞섰다.
**LINEMOD 데이터셋**에서는 거의 모든 물체에서 99% 이상의 완벽한 성능을 달성했으며, 평균 99.9%의 ADD-0.1d를 기록했다.
### 모델 기반 설정 성능
BOP 챌린지의 3개 핵심 데이터셋에서 평균 83.3%의 AR 점수를 달성했다. 이는 기존 새로운 물체를 다루는 모델 기반 방법들과 심지어 인스턴스 수준 방법들까지 큰 차이로 앞서는 성능이다.
## 4.3. 자세 추적 결과
### YCB-InEOAT 데이터셋
급작스러운 평면 밖 회전, 동적 외부 가림, 분리된 카메라 움직임의 도전을 포함하는 데이터셋에서 96.42% ADD-S와 93.09% ADD로 최고 성능을 달성했다
### YCB-Video 데이터셋
21개 물체에 대한 종합적인 비교에서 FoundationPose는 96.0% ADD와 97.9% ADD-S의 뛰어난 성능을 달성했다.
## 4.4. 상세 분석
### 절제 연구 결과
핵심 설계 선택들의 절제 연구<sub>ablation study</sub>를 YCB-Video 데이터셋의 AUC of ADD와 ADD-S 지표로 수행했다. 제안된 방법이 기본 버전으로 모델 프리 설정에서 16개 참조 이미지를 사용했다.
**LLM 질감 증강 제거** 시 성능이 약간 감소했고, **트랜스포머 대신 컨볼루션과 선형 층 사용** 시에도 비슷한 정도의 성능 하락을 보였다. 가장 큰 영향을 미친 것은 **계층적 비교의 제거**였다. 이 경우 렌더링과 자른 입력만을 자세 조건화된 심중향 손실<sub>pose-conditioned triplet loss</sub>로 학습하여 비교했다. 이로 인해 ADD에서 1.47%p, ADD-S에서 0.73%p의 상당한 성능 하락이 있었다. **InfoNCE 손실 사용** 변형도 대비 검증 쌍별 손실<sub>contrast validated pair-wise loss</sub>보다 성능이 떨어졌다. 이는 완벽한 평행이동 가정<sub>perfect translation assumption</sub> 때문으로 분석된다.
### 참조 이미지 수의 영향
참조 이미지 수가 결과에 미치는 영향을 YCB-Video 데이터셋의 AUC of ADD와 ADD-S로 연구했다. FoundationPose는 참조 이미지 수에 대해 강건하며, 특히 ADD-S 지표에서 그렇다. 두 지표 모두 12장에서 포화되는 경향을 보인다. 주목할 점은 단 4장의 참조 이미지만 제공되어도 16장의 참조 이미지를 사용하는 FS6D(88.4%)보다 여전히 강한 성능을 보인다는 것이다. 새로운 물체에 대해 많은 참조 이미지를 수집하는 것은 번거로울 수 있는데, 적은 수의 이미지로도 신뢰할 만한 성능을 보장할 수 있다면 실용성이 크게 증가한다.
### 학습 데이터 규모 법칙
이론적으로는 학습을 위한 합성 데이터를 무한정 생성할 수 있다. 학습 데이터 양이 결과에 미치는 영향을 YCB-Video 데이터셋의 AUC of ADD와 ADD-S 지표로 연구했다. 결과는 약 1M까지 이득이 포화됨을 보여준다. 이는 현재 데이터셋 크기가 충분하며, 더 많은 데이터를 생성하는 것보다는 데이터 품질이나 다양성 향상이 더 중요할 수 있음을 시사한다.
하지만 1M까지의 지속적인 성능 향상은 대규모 합성 학습의 중요성을 입증한다. 기존 방법들이 실제 데이터 부족으로 어려움을 겪었다면, 이 연구는 고품질 합성 데이터 생성을 통해 이 문제를 효과적으로 해결했다고 볼 수 있다.
### 계산 효율성
Intel i9-10980XE CPU와 NVIDIA RTX 3090 GPU에서 실행 시간을 측정했다. **자세 추정**<sub>pose estimation</sub>은 한 물체당 약 1.3초가 소요되며, 이는 자세 초기화 4ms, 정제 0.88초, 자세 선택 0.42초로 구성된다. **추적**<sub>tracking</sub>은 자세 정제만 필요하고 여러 자세 가설이 없기 때문에 훨씬 빠른 ~32Hz에서 실행된다.
실제로는 자세 추정을 한 번 수행하여 초기화하고 추적모드로 전환하여 실시간 성능을 달성할 수 있다. 이런 계산 효율성은 실제 로봇 응용에서 매우 중요한 요소이다. 실시간 추적 능력은 동적 환경에서의 실시간 물체 조작을 가능하게 한다.
# 5. Conclusion
## 5.1. 주요 기술적 성과
FoundationPose는 6D 자세 추정과 추적 분야에서 여러 중요한 돌파구를 달성했다. 가장 핵심적인 성과는 4가지 작업 조합 (model-based/free × estimation/tracking)을 단일 모델로 해결하는 최초의 통합 프레임워크를 구축했다는 것이다. 이는 기존의 분절된 접근법들과는 완전히 다른 패러다임을 제시한다.
둘째, 새로운 물체에 대해 미세 조정 없이 즉시 적용 가능한 진정한 **기반 모델**<sub>foundation model</sub>을 실현했다. 기존 방법들이 대상 데이터셋에서 미세 조정이나 적응이 필요했다면, FoundationPose는 사전 학습된 모델만으로 뛰어난 성능을 보장한다. 이는 강력한 일반화 능력을 보여준다.
셋째, 각 작업별로 특화된 기존 방법들을 상당한 차이로 앞서는 최고 수준 성능을 달성했다. 더욱 놀라운 것은 인스턴스 수준 방법들과 비교할 만한 결과를 더 적은 가정으로 실현했다는 점이다. 마지막으로 BOP 리더보드에서 1위를 달성하여 실제 응용에 바로 활용 가능한 수준임을 입증했다.
## 5.2. 기술적 혁신 요약
이 연구의 기술적 혁신은 여러 측면에서 중요하다. **LLM 지원 데이터 생성**은 ChatGPT와 TexFusion을 결합하여 현실적인 질감을 대규모로 생성하는 혁신적인 방법이다. 기존의 무작위 질감 혼합이 가졌던 한계를 완전히 극복하고 완전 자동화된 파이프라인을 구축했다.
**신경 암시적 표현**을 통한 모델 기반과 모델 프리 설정의 연결은 매우 우아한 해결책이다. SDF 기반 표현은 NeRF보다 더 정확한 깊이 렌더링을 제공하면서도 효율적인 새로운 시점 합성을 가능하게 한다. **계층적 자세 선택**은 전역 맥락을 고려한 2단계 비교 전략으로, 기존의 절대 점수 매기기 방식의 한계를 극복했다.
**분리된 자세 업데이트**는 평행이동과 회전을 카메라 프레임에서 독립적으로 처리하여 학습을 단순화하고 성능을 향상시켰다. 이런 여러 기술적 구성 요소들이 조화롭게 결합되어 전체적인 성능 향상을 이뤄냈다.
## 5.3. 한계점과 향후 연구
현재 FoundationPose는 몇 가지 한계점들을 가지고 있다. 가장 중요한 제약은 외부 2D 검출 모듈에 의존한다는 것이다. CNOS나 Mask R-CNN 같은 기성 방법들을 사용하는데, 잘못된 검출이나 누락된 검출이 전체 성능의 병목이 되는 경우가 많다. 이는 전체 파이프라인의 강건성을 제한하는 요소이다.
또한 매우 도전적인 조건에서는 여전히 실패하는 경우가 있다. 질감이 없는 물체에 심한 가림과 제한된 에지 단서가 동시에 발생하는 상황에서는 현재 방법으로도 올바른 방향을 추정하기 어렵다. 마지막으로 단일 강체 물체만 처리할 수 있어서 복잡한 관절 물체나 변형 가능한 물체는 다룰 수 없다.
향후 연구 방향으로는 검출부터 자세 추정까지의 **종단간 통합**<sub>end-to-end integration</sub>이 가장 중요하다. 새로운 물체 검출, 6D 자세 추정, 추적을 통합하는 프레임워크가 구축된다면 실제 응용에서의 강건성이 크게 향상될 것이다. 또한 여러 물체, 관절 구조, 장면 역학을 포함하는 장면 수준 이해로의 확장도 흥미로운 방향이다.
실제 배포를 위해서는 더 도전적인 실제 조건에 대한 강건한 처리 능력도 개선되어야 한다. 이 논문은 6D 자세 추정 분야에서 패러다임 전환을 제시하며, 기반 모델 접근법의 강력함을 입증한 중요한 연구로 평가된다.
